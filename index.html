Research

I am interested in low-level vision, computational photography, image compression, and invertibility problems.

European Conference on Computer Vision (ECCV), 2022
arXiv / bibtex / code

International Conference on Computer Vision (ICCV), 2021
arXiv / bibtex / code / presentation / supp / supp video
